# Overview
LaunchAPP is a Fire TV utility app that lets the user select an installed app and automatically launches that selected app after device boot (power on or restart). We take privacy seriously and designed LaunchAPP to work without collecting or selling personal data.

Data We Collect
LaunchAPP does not collect, transmit, or sell personal information.
We do not require an account, and we do not collect names, email addresses, phone numbers, precise location, or identifiers for advertising.

Data processed on device
To provide the app’s functionality, LaunchAPP may access and process the following locally on your device:

List of installed applications (app labels and package names) to allow you to select which app should be launched at boot.
App configuration settings stored locally (e.g., the selected app’s package name).
This information is used only to provide the requested features and is not sent to any server.

How We Use Data
We use on-device data only to:

display installed apps so you can choose one
save your selection on the device
launch the selected app automatically after boot
Data Sharing
We do not share data with third parties.
We do not integrate third-party analytics SDKs, advertising networks, or trackers.

Internet / Network Access
LaunchAPP does not require internet access for its core features.
If your build includes network permissions in the future, this policy will be updated accordingly.

Permissions
LaunchAPP may request the following permissions depending on your build/device:

RECEIVE_BOOT_COMPLETED: used to run after the device boots in order to launch the selected app automatically.
SYSTEM_ALERT_WINDOW (optional / only if used): if present, it is intended for features that require drawing over other apps. If you do not use overlay features, you should remove this permission from the app build.
Package visibility / queries: used to detect installed apps so they can be shown in the selection list.
Permissions are used strictly to provide the app’s functionality and are not used for tracking.

# Data Retention
Configuration data (the selected app package name) is stored locally on your device until you:

change the selected app, or
clear the app’s storage, or
uninstall LaunchAPP
# Security
We store configuration data using standard on-device storage provided by the Android/Fire OS platform. No user data is uploaded because the app does not operate a backend server for data collection.

# Children’s Privacy
LaunchAPP is not directed to children, and we do not knowingly collect personal data from children.
